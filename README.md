# race-cast-f1

**Race-Cast-F1** is a machine learning project in Python designed to forecast Formula 1 Grand Prix results using historical race, qualifying, and contextual data. 

The core of the project leverages feature engineering alongside **scikit-learn** and **XGBoost** models to learn performance patterns across drivers, teams, circuits, weather, and sessions. The repo provides scripts for data ingestion, preprocessing, model training with cross-validation, hyperparameter tuning, and performance evaluation, as well as utilities to generate and visualize prediction results. 

**Race-Cast-F1** is personal project born with the aim to both combine hands on Machine Learning tools in Python and my passion for F1.
